# bkmarker

A Social Bookmarking site written in Clojure.



## Set Up

1. Change database settings in project.clj and src/bkmarker/db/dbconn.clj
2. Create database
3. Migrate database with: `lein ragtime migrate`
4. start the server `lein run`
